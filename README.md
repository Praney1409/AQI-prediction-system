# 🌍 AQI Prediction System

An AI-powered Air Quality Index (AQI) Prediction System that forecasts air pollution levels using Machine Learning techniques. The system analyzes environmental parameters and predicts future AQI values to help users monitor air quality and make informed decisions.

---

## 🚀 Features

* 📊 Predict Air Quality Index (AQI)
* 🤖 Machine Learning-based Forecasting
* 📈 Data Visualization and Analysis
* 🧹 Data Preprocessing and Feature Engineering
* 📉 Performance Evaluation Metrics
* 🌐 User-Friendly Interface
* 📋 Historical AQI Trend Analysis

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AQI-Prediction-System.git
cd AQI-Prediction-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
